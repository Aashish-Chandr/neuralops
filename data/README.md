# Training Data

Generated by `scripts/generate_training_data.py`. Do not edit manually.

| File | Shape | Description |
|---|---|---|
| `normal_sequences.npy` | (5000, 60, 5) | Normal operating metric sequences |
| `anomaly_sequences.npy` | (1000, 60, 5) | Anomalous sequences (NOT used for training) |
| `normal_sequences_norm.npy` | (5000, 60, 5) | Normalized normal sequences |
| `norm_stats.json` | — | Min/max per feature for normalization |
| `sample_metrics.csv` | 6000 rows | Last timestep of each sequence, for inspection |
| `metadata.json` | — | Feature names, shapes, generation params |

## Features (per timestep)
1. `cpu_usage_percent` — CPU utilization %
2. `memory_usage_percent` — Memory utilization %
3. `latency_p99_ms` — P99 request latency in ms
4. `error_rate_percent` — HTTP error rate %
5. `requests_per_second` — Throughput

## Regenerate
```bash
python scripts/generate_training_data.py
```
